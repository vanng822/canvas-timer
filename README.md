## Canvas timer

Time's up using canvas, support pie-timer and bar-timer

## Example
	var timer = new CanvasPieTimer(1000, 30000, 26, 26,  function() {
		console.log('Time to party');
	});
	
	var timer = new CanvasBarTimer(1000, 30000, 100, 26,  function() {
		console.log('Time to party');
	});
